JS基础（二）数据转换和运算符

2019-07-29

一、数据类型

JS的数据类型主要分为基本类型和引用类型

1、数据类型

  • JS数据类型分为两种：
  •  基本数据类型：Number、String、Boolean、Null、 Undefined

  •  引用数据类型：object

2、栈和堆

  •  栈为自动分配的内存空间，它由系统自动释放；堆动态分配的内存，大小不一，需要手动释放

二、数据类型转换

1、字符串转换为数值

//parseInt（）、Number（）把字符串转换为数值类型
        var str1="aaa";
        console.log(parseInt(str1));//NaN  非数值，强制了数据类型是数值型，但是值并不是数值，这时候就是NaN
        console.log(Number(str1));//NaN  非数值，强制了数据类型是数值型，但是值并不是数值，这时候就是NaN
        var str2="38";
        console.log(parseInt(str2)); //38
        console.log(Number(str2)); //38
        var str3="38a";
        console.log(parseInt(str3));//38;
        console.log(Number(str3));//NaN; 使用Number强转，如果内容有字符，就直接判定为NaN
        var str4="a38";
        console.log(parseInt(str4));//NaN
//parseInt（）与Number（）区别主要是当前面是数字后面是字符parseInt抛弃后面字符，但Number直接转换为NaN;若开始就是字符，直接转换为NaN
/*
*String();     转换为字符串
*toString();   按进制转换
*toFixed();     保留几位小数，自动四舍五入
*/

三、运算符

JS运算符主要包括算数运算符、一元运算符、关系运算符、逻辑运算符、赋值运算符、位运算符

1、算数运算符

//赋值    =
var x=1;
var y=2;
var z=x+y;
console.log(z);

//对于字符串相加---首位相连
var str1="a";
var str2="b";
console.log(str1+str2);

//隐式转换

数值、布尔,undefined，null加字符串 ----
将数值、布尔、undefined、null转换为字符串进行字符串拼接

数值与布尔----
先将布尔转换为数值类型，true为1，false为0；
null转换为0；undefined由于转换之后为NaN，所以数值与undefined进行算数运算结果都为NaN

undefined----
除了与字符串做加法进行拼接，其余都是NaN

字符串、布尔、undefined、null与数值进行除加法之外的运算都是先把它们转换为数值进行求值。

2、一元运算符

var x=5;
var y=x++;
console.log(y,x);//先进行赋值运算把  x=5赋值给y,然后进行x加一
//y=5;x=6

var x=5;
var y=++x;
consoele.log(y,x)//先进行x加一，然后把x赋值给y
//y=6;x=6

4、逻辑运算符

//逻辑运算符结果为布尔值
//取反  ！a
var a;
if(a){
	//a不是0、false、“”、null、undefined、NaN就满足
}

if(!a){
	//a是0、false、“”、null、undefined、NaN就满足
}

5、赋值运算符

var num =5;
num+=5; //num=num+5;
console.log(num);
var num1="5";
num1+=2;
console.log(num1); //52(字符串)；

		*   -=
		*  +=
		*   *=
